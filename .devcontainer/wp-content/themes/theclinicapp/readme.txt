TheClinicApp is a custom WordPress theme built from scratch using Bootstrap 5. It is designed to be clean, responsive, and developer-friendly. 

== Description ==
This theme was built to replicate the structure and styling of the static website TheClinicApp (https://animocdn.co/TCA/git/), converted to a WordPress classic theme using Bootstrap 5. It uses custom template files, a separate header and footer, and includes a theme screenshot and font assets.
TheClinicApp is a patient and practice management software app.

== Installation ==
1. Upload the `theclinicapp` theme to your `/wp-content/themes/` directory.
2. Activate the theme from the WordPress dashboard under Appearance > Themes.
3. Customize as needed.
